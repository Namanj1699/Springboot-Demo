# Spring Boot Demo

This project is part of my Spring Boot learning and upskilling journey. I recently started learning Spring Boot and building backend applications using Java 21 and the latest Spring Boot annotations and features.

The project demonstrates complete CRUD (Create, Read, Update, Delete) operations using Spring Boot, Spring Data JPA, and REST APIs. It is designed to help me understand backend development concepts, database integration, entity relationships, validations, and API development.

## Technologies Used

* Java 21
* Spring Boot
* Spring Data JPA
* Hibernate
* MySQL
* Maven

## Features

* Create records
* Fetch records
* Update records
* Delete records
* REST API implementation
* Usage of latest Spring Boot annotations
* Layered architecture (Controller, Service, Repository)
